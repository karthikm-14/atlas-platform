# atlas-platform

Atlas is a scalable backend platform designed to support AI-based SaaS products. 


## Overview

Atlas is a production-ready backend built with FastAPI and PostgreSQL, focusing on
clean architecture, secure authentication, and AI-ready extensibility.


## Features

- JWT-based authentication
- Role-based access control
- Modular API architecture
- Dockerized local development
- Centralized error handling

> Future: AI workflows, embeddings, vector search
